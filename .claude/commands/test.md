# اجرای کامل تست

```bash
go vet ./... && go test -race -cover ./...
```

اگر هر کدام شکست بخورند، خروجی را تحلیل کن و پیشنهاد اصلاح بده.
گزارش نهایی:
- تعداد تست‌های گذرانده/شکسته
- درصد پوشش هر پکیج
- هشدارهای `go vet`
